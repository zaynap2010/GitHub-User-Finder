# GitHub User Finder

## 👤 Автор
**Зайнап** 

## 📝 Описание
GitHub User Finder — десктопное GUI-приложение для поиска пользователей GitHub. Позволяет находить информацию о разработчиках, просматривать их профили и сохранять в избранное.

## 🔧 Как использовать API
Приложение использует публичный REST API GitHub:
- **Эндпоинт:** `GET https://api.github.com/users/{username}`
- **Пример ответа:** логин, имя, количество репозиториев, подписчики, подписки

## 📖 Примеры использования
1. Введите логин пользователя (например, `octocat`)
2. Нажмите "Найти" или Enter
3. Нажмите "⭐ Добавить в избранное" для сохранения
4. Перейдите на вкладку "Избранное" для просмотра сохранённых пользователей
